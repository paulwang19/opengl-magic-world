## 簡介

設計一個魔法世界，裡面有渲染材質，且具有動畫的物件

## 功能要求

- 使用 4 種以上以上不同材質與物件
- 至少載入一個具有動畫，且與未來世界主題有關的物件
- 一個繞著某個軸線公轉的 A 物件
- 一個繞著 A 物件公轉的 B 物件，且呈現 cos-like 波型
- 製作一個與前述不同的動畫，應用在 C 物件上
- 所有物件都有影子
